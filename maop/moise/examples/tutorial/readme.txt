This directory contains some moise+ examples used in the moise+ tutorial.


1. CLASSPATH

   unix:
     export CLASSPATH=../../lib/moise.jar:.

   windows:
     set CLASSPATH=..\..\lib\moise.jar;.



2. Structure

   Compilation: javac TutorialSS.java
   Execution:   java  TutorialSS


3. Functioning

   Compilation: javac TutorialFS.java
   Execution:   java  TutorialFS


3. Normative

   Compilation: javac TutorialDS.java
   Execution:   java  TutorialDS
